Populations - Single State

Return the population of a given state.

Request URL

GET http://api.data.gov/census/american-community-survey/v1/{year}/populations/{state}?parameters

Request Parameters

Parameter Required Value Description
year Yes
Type: integer
Default: None
Options: 2010, 2011

The American Community Survey year to pick.
state Yes
Type: string
Default: None
Options: json, xml

Two letter USPS state code of the state.
api_key Yes
Type: string
Default: None

Your developer API key. See API keys for more information.

Examples

JSON Output Format

GET /census/american-community-survey/v1/2011/populations/NY?api_key=DEMO_KEY
[
  ["B02001_001E","NAME","state"],
  ["19302448","New York","36"]
]

Rate Limits

Standard rate limits apply. No more than 1,000 requests may be made in any hour. No more than 10,000 requests may be made in any day.

Errors

Standard errors may be returned.
